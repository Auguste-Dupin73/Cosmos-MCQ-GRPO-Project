//detect_duplicates.py
How to use it
1) Just detect duplicates (and treat lists as unordered)
python detect_duplicates.py --in_seeds example-generator-v1/data/team_seeds/tierB/codex_B.jsonl --ignore_list_order

2) Detect duplicates within each template only
python detect_duplicates.py --in_seeds seeds.jsonl --ignore_list_order --template_scoped

3) Write duplicates to quarantine file
python detect_duplicates.py --in_seeds seeds.jsonl --ignore_list_order --quarantine dups.jsonl

4) Produce a de-duplicated file (keep first occurrence)
python detect_duplicates.py --in_seeds seeds.jsonl --ignore_list_order --dedupe --out_seeds seeds.deduped.jsonl

5) De-duplicate but keep the last occurrence
python detect_duplicates.py --in_seeds seeds.jsonl --ignore_list_order --dedupe --keep last --out_seeds seeds.deduped.jsonl
